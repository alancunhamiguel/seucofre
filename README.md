SeuCofre - Cloud Storage Solution

O **SeuCofre** é um protótipo funcional de uma plataforma de armazenamento de arquivos na nuvem. 
Este projeto foi desenvolvido como parte do meu portfólio profissional para demonstrar habilidades 
em desenvolvimento Full-Stack, autenticação segura e integração com serviços de terceiros.

---

Demonstração

O projeto está hospedado e pronto para testes:
*   **Site Pronto:** [https://seucofrebr.onrender.com](https://seucofrebr.onrender.com)

---

Tecnologias Utilizadas

O ecossistema do projeto utiliza a **MERN Stack** e outras ferramentas modernas:

- **Frontend:** React.js, Axios, CSS3 (Design Responsivo).
- **Backend:** Node.js, Express.js.
- **Banco de Dados:** MongoDB Atlas (NoSQL).
- **Armazenamento de Arquivos:** Cloudinary API.
- **Autenticação:** JSON Web Tokens (JWT) e Bcrypt.js para hashing de senhas.
- **Deploy:** Render (Web Services & Static Sites).

---

Funcionalidades Principal

- **Autenticação de Usuários:** Cadastro e login com proteção de rotas.
- **Upload de Arquivos:** Integração direta com a nuvem via Cloudinary.
- **Gestão de Espaço:** Limite inteligente de 2GB por conta de usuário.
- **Listagem em Tempo Real:** Visualização dinâmica dos arquivos enviados.
- **Exclusão Segura:** Remoção de registros no banco de dados e sincronização com o storage.
